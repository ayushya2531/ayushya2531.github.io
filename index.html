Projects

Learning Methods for Nearfield Beamforming

The aim of the project was to improve upon existing range estimation techniques through deep learning based approaches to benefit near-field beamforming. Spherical Harmonic features were used after, in a learning framework, for range estimation of a signal source. This choice of using Spherical Harmonic features was made after testing on what different feature sets could be used for training the network. Near-field Beamforming experiments run according to the developed technique for range estimation were found to give an improved response. A paper has been submitted to the International Conference on Acoustics, Speech and Signal Processing, 2018. We are working on an advanced implementation of the technique and thus the draft and the code have not been released.

Human Pose Estimation and Segmentation

Read Full Paper

The aim of the project was to develop an end to end fully convolutional model for joint human pose point estimation and human part segmentation via hard parameter sharing. We created a flexible pipeline for joint learning of the two complementary tasks, inspired by the CVPR'17 paper on pose estimation using Part Affinity Fields and the standard FCN architecture. The model was implemented in Pytorch and experiments were run on MPII and PASCAL-VOC part dataset to verify the compatibility of the two tasks.

Multi-task learning through Probabilistic SVMs

Read Full Paper -Github

The aim was to formulate an algorithm for the utilization of Bayesian Support Vector Machines in a Multi-Task Learning framework that considered multiple related binary classification tasks. The problem was modeled generatively using latent variables describing the task cluster identity, the cluster dependent weight priors and the latent variables of the Bayesian SVM model. We utilized the EM algorithm along with Gibbs sampling to train the model and got improved accuracy over single learning tasks. Game Playing Bot

The aim was to implement a bot that learned to play a game using Reinforcement Learning in an OpenAI environment. We implemented DeepMind's 2015 paper "Human Level Control through deep reinforcement learning" in the context of learning to play Atari Games and further, trained and tested the implemented module on the Atari game, Breakout, whose environment was provided by OpenAI gym.

Prediction of Words from fMRI images

Read Full Paper

The aim was to implement different ML models to classify the fMRI images taken when an individual was being shown a drawing representing the word. I explored the works of the 'Brain Image Analysis Research Group' of CMU and utilized the data provided by them, implemented different feature reduction techniques and models for classification with the best performance being given by using a linear mapping of image features to word semantics and performing classification in the mapped space
